Project 01
Pair Project
Dragonbrew

Dragonbrew was our very first project during XAMK university studies. Made in Unity in collaboration with Michaela Heikkonen, the two of us split responsibility roughly evenly across design, programming, and 2D art.

As a first proper game project at university level, Dragonbrew was as much about learning to work as a pair on a shared codebase as it was about the game itself. Getting something playable and cohesive from a two-person team under time pressure taught me a lot about scoping, communication, and when to stop adding features.

Game Design (50%) Programming (50%) 2D Art (50%)

Project 02
Solo Project
The Little Mole Platformer

A small solo project focused on practising 2D animation and platformer mechanics in Unity. Every piece of this one is mine: all the visuals, programming, and design.

The art was drawn in Aseprite in pixel art style, referencing the classic Czech cartoon character Krteček (The Little Mole) by Zdeněk Miler, first created in 1956. Working from a reference character with such a strong established visual identity meant staying faithful to the design while translating it into pixel art, which required its own set of decisions.

This project was primarily about getting comfortable with 2D animation systems in Unity and understanding how visual feedback and movement feel connect in a platformer.
